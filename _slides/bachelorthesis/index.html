# Ergebnisse
#### Standard ## HL7 FHIR * deckt viele Anwendungfälle ab * flexibel über Extensions * leicht zu integrieren * basiert auf bekannten Webtechnologien
#### Plattform ## iOS HealthKit * aggregiert gesundheitsrelevante Daten * bietet Schnittstelle zum Austausch * Zugriff auf Samples muss einzeln freigegeben werden
#### Ökosystem ## Javascript * React Native für mobile Applikation * Nodejs für Server * React für Frontend
